### Animal Jam Classic Socket Bots

AJCBots is a socket-based bot framework for **Animal Jam Classic**, intended for developers and researchers who require low-level access to the game‚Äôs network protocol.

This project enables direct communication with the game servers without the official client, allowing for automation, testing, and protocol research.

## üîå Overview

AJCBots connects to Animal Jam Classic servers using raw socket connections, providing full control over packet handling and client behavior.

The project is designed for advanced use cases where direct packet interaction is required.

---

## ‚ú® Features

- **Direct socket communication**
  - Connects to Animal Jam Classic servers without using the official client.
  - Allows full control over sent and received packets.

- **Bot-driven interaction**
  - Send and receive chat messages programmatically.
  - Implement automated responses, commands, or monitoring systems.

- **Packet-level access**
  - Suitable for reverse engineering and research analysis.
  - Enables experimentation with packet structures and server behavior.

- **Extensible design**
  - Easily expandable with new handlers, features, or logic.
  - Designed to support multiple bots and custom workflows.
